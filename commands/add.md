## git add

---
Синтаксис: 
```bush-
git add <file name>
```

Команда **`git add`** добавляет содержимое рабочего каталога в индекс (staging area) для последующего коммита.

Ниже приведен код добавления всех изменений в индекс (кроме игнорируемых .gitignore):

```bush-
git add .
```
